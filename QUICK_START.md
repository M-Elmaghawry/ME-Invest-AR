# ⚡ دليل البداية السريع - Quick Start

## 🚀 في 3 خطوات فقط!

### 1️⃣ افتح الموقع
```bash
# افتح index.html في المتصفح
start index.html
```

### 2️⃣ شاهد البيانات
- 🟡 أسعار الذهب (مصر - السعودية)
- 💵 سعر الدولار
- 🔁 محول العملات

### 3️⃣ تفاعل
- غيّر الدولة (مصر/السعودية)
- غيّر العيار (21/24)
- جرب محول العملات
- اضغط على فترات الرسم البياني

---

## 🎯 مثال سريع

### مثال 1: كم سعر جرام الذهب عيار 21 في مصر؟
```
1. افتح الموقع
2. تأكد من اختيار "مصر"
3. اختر "عيار 21"
4. شاهد السعر الحالي
```

### مثال 2: كم يساوي 100 دولار بالريال؟
```
1. انزل لقسم محول العملات
2. اكتب 100 في خانة USD
3. شاهد القيمة تلقائيًا في SAR
```

### مثال 3: كيف كان سعر الدولار قبل شهر؟
```
1. انظر لقسم الدولار
2. شاهد مؤشر "مقارنة مع شهر"
3. أو شاهد الرسم البياني
```

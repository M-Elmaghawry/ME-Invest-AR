## [1.0.0] - 2026-02-01

### ✨ Added (إضافات)

#### 🟡 قسم الذهب
- عرض أسعار الذهب لمصر والسعودية
- دعم عيار 21 وعيار 24
- حساب تلقائي لعيار 21 من عيار 24
- مؤشرات مقارنة زمنية (أمس - أسبوع - شهر - 6 شهور - سنة)
- رسم بياني تفاعلي مع فترات زمنية متعددة (7 أيام - 30 يوم - سنة)
- ألوان ديناميكية حسب اتجاه السعر (أخضر/أحمر/رمادي)

#### 💵 قسم الدولار
- عرض سعر الدولار الأمريكي مقابل الجنيه المصري
- نفس المؤشرات الزمنية للمقارنة
- رسم بياني تاريخي للتتبع

#### 🔁 محول العملات
- تحويل فوري بين:
  - الجنيه المصري (EGP)
  - الريال السعودي (SAR)
  - الدولار الأمريكي (USD)
- تحديث تلقائي لجميع الحقول
- منع الحلقات اللانهائية في التحويل

#### 🎨 التصميم
- تصميم عصري واحترافي
- دعم كامل للغة العربية واتجاه RTL
- تصميم متجاوب (Responsive) لجميع الشاشات
- ألوان متسقة مع نظام التصميم
- رسوم متحركة سلسة (Smooth animations)
- أيقونات emoji للدول

#### 📊 البيانات
- ملف JSON تاريخي (`market-history.json`)
- تخزين بيانات آخر 365 يوم
- هيكل بيانات موحد وقابل للتوسع

#### 🤖 الأتمتة
- GitHub Actions للتحديث التلقائي اليومي
- سكريبت Python لجلب الأسعار
- Commit تلقائي للتحديثات
- دعم التشغيل اليدوي (Manual trigger)

#### 📚 التوثيق
- `README.md` - دليل شامل للمشروع
- `API_GUIDE.md` - شرح مفصل لإعداد APIs
- `TECHNICAL_DOCS.md` - توثيق فني للمطورين
- `DEPLOYMENT.md` - دليل النشر والاستضافة
- `CHANGELOG.md` - سجل التغييرات (هذا الملف)

### 🔧 Technical Features

- **Frontend Only**: لا يتطلب Backend
- **Static Site**: سريع وآمن
- **Chart.js Integration**: رسوم بيانية تفاعلية
- **Vanilla JavaScript**: بدون dependencies ثقيلة
- **CSS Variables**: سهولة التخصيص
- **GitHub Pages Ready**: جاهز للنشر مباشرة

### 🌍 Localization

- واجهة بالعربية بالكامل
- تنسيق التواريخ بالعربية
- دعم RTL كامل
- أرقام عربية في بعض العناصر
